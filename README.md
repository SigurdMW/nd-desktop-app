# Norsk Display Desktop App
Desktop app using: 
* Electron
* Lit and lit-html (web components)
* TailwindCSS
* Serialport

## Develop
Start: `npm start`
Build: `npm run build`

## Build and Package
`npm run package`
Use RC Edit to rename and brand the application: https://github.com/electron/rcedit
